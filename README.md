DESCRIPCIÓN

 * GUÍA RÁPIDA: ¿QUÉ HACE CADA FUNCIÓN Y QUÉ DEVUELVE?
PARTE 1
ft_isalpha — ¿es una letra? Devuelve: 1 si lo es, 0 si no.
ft_isdigit — ¿es un dígito (0-9)? Devuelve: 1 / 0.
ft_isalnum — ¿es letra o dígito? Devuelve: 1 / 0.
ft_isascii — ¿está en el rango ASCII (0-127)? Devuelve: 1 / 0.
ft_isprint — ¿es imprimible? Devuelve: 1 / 0.
ft_strlen — longitud de la cadena. Devuelve: size_t, sin contar el \0.
ft_memset — rellena n bytes con un valor. Devuelve: puntero al bloque (void *mem).
ft_bzero — pone n bytes a cero. Devuelve: nada (void).
ft_memcpy — copia n bytes. Devuelve: puntero a dst.
ft_memmove — copia n bytes, seguro con solapamiento. Devuelve: puntero a dst.
ft_strlcpy — copia cadena con límite de buffer. Devuelve: size_t, longitud de src.
ft_strlcat — concatena con límite de buffer. Devuelve: size_t, longitud total que intentó crear.
ft_toupper — a mayúscula. Devuelve: int, el carácter convertido.
ft_tolower — a minúscula. Devuelve: int, el carácter convertido.
ft_strchr — primera aparición de un carácter. Devuelve: puntero a esa posición, o NULL.
ft_strrchr — última aparición de un carácter. Devuelve: puntero, o NULL.
ft_strncmp — compara dos cadenas hasta n bytes. Devuelve: int: 0 si iguales, <0 o >0.
ft_memchr — busca un byte en n bytes de memoria. Devuelve: puntero al byte, o NULL.
ft_memcmp — compara n bytes de memoria. Devuelve: int, la diferencia.
ft_strnstr — busca una subcadena dentro de los primeros len bytes. Devuelve: puntero al inicio, o NULL.
ft_atoi — convierte cadena a entero. Devuelve: int, el número.
ft_calloc — reserva memoria inicializada a 0. Devuelve: puntero a la memoria, o NULL.
ft_strdup — duplica una cadena. Devuelve: puntero a la copia nueva, o NULL.
